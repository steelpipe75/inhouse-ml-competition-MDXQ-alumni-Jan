# 内輪向け 機械学習コンペ用 JupyterLite 環境

このリポジトリは、**内輪向けの機械学習コンペ**に参加するための  
**ブラウザ完結型 Jupyter Notebook 実行環境（JupyterLite）** です。

Python の実行環境構築やライブラリのインストール不要で、  
ブラウザを開くだけですぐに分析・モデル作成を始められます。

---

## この環境でできること

- ブラウザ上で Python Notebook を実行
- コンペ用の **train / test データ**を読み込み
- **サンプルノートブック**を元にモデル作成・改善
- 予測結果ファイル（CSV）をローカルにダウンロード
